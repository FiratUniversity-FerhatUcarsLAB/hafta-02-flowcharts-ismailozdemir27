1. Genel Yapı – Sistem Akışı
Bir üniversite ders kayıt sistemi şu adımlardan oluşur:

Giriş (Authentication):
Öğrenci kullanıcı adı ve şifreyle sisteme giriş yapar.

Ders Listesi Gösterimi:
Öğrenciye, dönem içinde açılmış ve kendisinin alabileceği derslerin listesi sunulur.

Ders Seçimi:
Öğrenci almak istediği dersleri seçer (çoklu seçim yapılabilir).

Kontrollerin Yapılması:
Seçilen her ders için aşağıdaki kontroller yapılır:

Kontenjan var mı?

Ön koşul sağlanmış mı?

Zaman çakışması var mı?

Toplam kredi limiti aşılmış mı?

Danışman Onayı:
Tüm kontrollerden sonra danışmanın onayına gönderilir.

Onaylama ve Kayıt:
Danışman onayladıktan sonra kayıt tamamlanır.

2. Kontrol Mekanizmaları
Her ders seçimi için sistem şu kontrolleri yapmalıdır:

Kontenjan Kontrolü:
Dersin kalan kontenjanı var mı?

Ön Koşul Kontrolü:
Dersi alabilmek için gerekli olan önceki ders(ler) alınmış mı?

Zaman Çakışması Kontrolü:
Seçilen dersler aynı gün/saatte mi?

Kredi Limiti Kontrolü:
Öğrencinin seçtiği derslerin toplam kredisi maksimum limiti aşıyor mu?

Sistem kontrolleri geçse bile, kayıt danışman onayı olmadan tamamlanamaz.

4. DOT Dönüşümü (Graphviz DOT formatında Akış Diyagramı)
Her kontrol noktasını diamond (elmas) şekli ile gösteriyoruz.

digraph DersKayit {
    node [shape=rectangle]

    Giris [label="Giriş Yap"]
    GirisKontrol [label="Giriş Başarılı mı?", shape=diamond]
    DersListesi [label="Ders Listesini Göster"]
    DersSecimi [label="Ders Seç"]
    KontenjanKontrol [label="Kontenjan Var mı?", shape=diamond]
    OnKosulKontrol [label="Ön Koşul Sağlanmış mı?", shape=diamond]
    ZamanCakismaKontrol [label="Zaman Çakışması Var mı?", shape=diamond]
    KrediKontrol [label="Kredi Limiti Aşıldı mı?", shape=diamond]
    DanismanOnay [label="Danışman Onayı?", shape=diamond]
    KayitTamam [label="Kayıt Başarılı"]
    KayitBasarisiz [label="Kayıt Başarısız"]
    Hata [label="Hata Mesajı Göster"]

    Giris -> GirisKontrol
    GirisKontrol -> DersListesi [label="Evet"]
    GirisKontrol -> Hata [label="Hayır"]

    DersListesi -> DersSecimi
    DersSecimi -> KontenjanKontrol

    KontenjanKontrol -> OnKosulKontrol [label="Evet"]
    KontenjanKontrol -> Hata [label="Hayır"]

    OnKosulKontrol -> ZamanCakismaKontrol [label="Evet"]
    OnKosulKontrol -> Hata [label="Hayır"]

    ZamanCakismaKontrol -> KrediKontrol [label="Hayır"]  // çakışma yoksa
    ZamanCakismaKontrol -> Hata [label="Evet"]

    KrediKontrol -> DanismanOnay [label="Hayır"] // limit aşılmamışsa
    KrediKontrol -> Hata [label="Evet"]

    DanismanOnay -> KayitTamam [label="Evet"]
    DanismanOnay -> KayitBasarisiz [label="Hayır"]
}
